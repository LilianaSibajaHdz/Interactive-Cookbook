# REDECOIN - Recetario de Cocina Interactivo

**REDECOIN** es una aplicación web interactiva diseñada para gestionar recetas personales de manera dinámica. Permite a los usuarios registrar nuevas preparaciones, visualizarlas en tarjetas detalladas, filtrar por nombre o categoría y mantener la información persistente.

El objetivo principal de este proyecto es consolidar los conocimientos de **JavaScript desde Cero** impartido en **DEV.F**, aplicando lógica de programación real y manipulación avanzada de la interfaz de usuario.

## 🚀 Objetivo del Proyecto

Aplicar de forma práctica conceptos fundamentales de desarrollo Front-End, incluyendo:
* **JavaScript básico** 
* **Estructuras de control** 
* **Funciones** 
* **Arreglos** 
* **Objetos** 
* **Manipulación del DOM** 
* **Almacenamiento en** `localStorage` 
* **Maquetación con Bootstrap** 
* **Buenas prácticas básicas de organización de código** 

## 🛠️ Tecnologías Utilizadas

* **HTML5**
* **Bootstrap**
* **Font Awesome**
* **DOM API**
* **Local Storage**
